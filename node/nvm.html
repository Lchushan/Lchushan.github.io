# nvm

# Mac 安装nvm

路径：https://blog.csdn.net/ForeverMyheart/article/details/127203419

# 第一种安装方案，手动下载

# 1、下载nvm

从github下载nvm (opens new window)仓库到 ~/目录 地址：https://github.com/nvm-sh/nvm.git (opens new window)

# 2、执行install.sh

进入 nvm目录中执行install.sh 等待执行完成，执行的操作方法就是直接将文件拖入终端然后回车。

# 3、配置nvm环境变量将下述代码复制到 ~/.bash_profile

export NVM_DIR="$HOME/.nvm"
[ -s "$NVM_DIR/nvm.sh" ] && \. "$NVM_DIR/nvm.sh" 
 # This loads nvm
[ -s "$NVM_DIR/bash_completion" ] && \. "$NVM_DIR/bash_completion"
# This loads nvm bash_completio

如果没有配置环境变量，要先配置环境变量 ，配置环境变量 (opens new window)的方法：

1.在终端输入touch ~/.bash_profile

2.在终端输入source ~/.bash_profile

3.在终端输入. ~/.nvm/nvm.sh

4.打开环境变量文件的方法：open ~/.bash_profile

# 4、执行source ~/.bash_profile

# 5、执行nvm --version是否可以正常输出，若不行则重启终端再次尝试

# 第二种安装方案，通过brew安装

# 1、brew install nvm

首先要保证之前没有安装过node，如果之前安装过，就先 brew uninstall noded

# 2、报错处理

上面的操作之后，终端可能会报错：Command failed with exit 128: git，不要着急，这是终端输入brew -v ，回提示添加git config --global --add safe.directory /usr/local/Homebrew/Library/Taps/homebrew/homebrew-core和git config --global --add safe.directory /usr/local/Homebrew/Library/Taps/homebrew/homebrew-cask,只要将两行代码复制到终端执行，之后再重新brew install nvm，之后就是等待了

# 3、下载后配置

下载完之后，需要在环境变量的文件中添加 echo "source $(brew --prefix nvm)/nvm.sh" >> .bash_profile，并source ~/.bash_profile保存变量，之后就可以正常使用了

# nvm操作

1：nvm install 版本号-- 安装指定node版本

2：nvm uninstall 版本号-- 删除指定node版本

3：nvm use 版本号 --切换使用指定的node版本

4：nvm list 或 nvm ls -- 列出所有安装的node版本

5：nvm current -- 显示当前的node版本

6：nvm alias default 版本号 -- 可以指定默认打开终端时的node版本
