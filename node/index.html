# nodejs

# 学习node.js的建议：

当前node.js学习阶段，是服务器端的开发，要把自己当作一名后端开发人员，用后端的思维去考虑问题。

虽然都是JavaScript这门脚本语言，但是承载的角色有所不同，要区别对待

我们之前写的项目最终都是运行在服务端的，要时刻建立一种请求，处理，响应的前后端交互思维

抓紧一切事件进行练习，以巩固听课效果

1.使用右边的方式，追加内容到文件内，如果没有这个文件，也会自动新建一个文件，但不会自动新建文件夹
